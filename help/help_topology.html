Topology Help


Network Topology


Display topology graph of the managed Access Points. It displays connectivity graph of managed Access Points. The Access Point icons could be moved on the topology background and their locations saved for later displays.


  • Background Image File: Specify floor Map jpg/gif image of size 800x600 to be uploaded for displaying as topology background.


Use the REFRESH button to update connectivity status.


Use the APPLY button to save the location of the Access Points on the floor displayed floor map.
