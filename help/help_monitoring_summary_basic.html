Monitoring Summary Help


Basic


This screen displays a read-only summary of the current managed Access Point status, rogue access points detected, current wireless stations connected, WMS5316 system Information and Network Usage.


Access Point Status


This section displays status of managed Access Points.


  • Total Configured: Total number of managed Access Points.

  • Down: Number of managed Access Points which cannot be pinged.

  • Critical: Number of managed Access Points which can be pinged but cannot be logged in or device is different from the one which was configured.

  • Major: Number of managed Access Points whose configuration differs from the one which is set on WMS5316. This is mostly owing to Access Point having an unsupported software version running or configuration changes were done on WMS when Access Point was Down/Offline.

  • Healthy: Managed Access Points with running properly.


Rogue Access Points


This section displays the count of rogue/neighboring access points discovered by managed Access Points.


  • Rogue AP current: The number of unique rogue/neighboring Access Point bssid which can be observed now.

  • Rogue AP count 24hrs: The number of unique rogue/neighboring Access Point bssid observed over the last 24 hrs.


Wireless Stations


This section displays count of Current Wireless Stations of managed Access Points.


  • open: Count of Wireless Stations connected to managed Access Points using security profiles configured with open mode.

  • wep: Count of Wireless Stations connected to managed Access Points using security profiles configured with WEP.

  • wpa: Count of Wireless Stations connected to managed Access Points using security profiles configured with WPA security.

  • wpa2: Count of Wireless Stations connected to managed Access Points using security profiles configured with WPA2 security.


Network Info


This section displays WMS5316 Information.


  • Last Channel Allocation Run: when last automatic channel Allocation was performed.

  • Last Configuration Change: when last configuration change was done on WMS5316.

  • Last WMS5316 Reboot: when was the WMS5316 rebooted last time.

  • Last Admin Login: when Admin logged in last time.

  • WMS5316 Firmware version: current WMS5316 firmware version.


Network Usage


This section displays average data traffic received and transmitted per managed Access Point over the last 24 hours.
