Access Points Remote Console Help


Remote Console


Enable SSH or Telnet to login into managed Access Points. These settings are only applied on managed Prosafe Access Points which support SSH and telnet.


Use the Group tab to select the settings for a specified group of Access Points. The Access Points which have not been assigned any group share the settings of Basic group.


  • Secure Shell (SSH): If set to Enable, Access Points will allow remote access by using Secure Shell.

  • Telnet: If set to Enable, Access Points will allow remote access by using Telnet.


Click APPLY button to save the selected settings
