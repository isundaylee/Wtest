Basic Profile Settings Help


Profile Settings

Details of each Wireless Network are contained in a Security Profile. This page lets you edit up to eight Security Profiles for managed Access Points in default group. The number of security profiles applied to managed Access Point depends on supported profiles on Access Point. Separate profiles are applied to 802.11 b/bg/ng and 802.11a/na mode radios.


Each Security Profile specifies:

  • Profile Name: Displays the unique Profile name. This value can be up to 31 alphanumeric characters.

  • SSID: Displays the current SSID associated with this profile.

  • Security: Displays the current security standard, such as WPA-PSK, associated with the profile.

  • VLAN: Displays the VLAN ID associated with this security profile.


Select the Enable check box to enable (or disable) the corresponding profile.


To change the settings of a security profile, select the profile and click the Edit button.
