Managed Access Points Edit Info Help


This page allows editing individual Access Points settings of managed Access Points.


Access Points Info

  • Name: The unique name of managed Access Point.

  • Model: Displays the model of Access Point. Field cannot be modified. It is determined at the time of adding to Managed List.

  • User Name: Displays login name for login to Access Point. This field is not modifiable.

  • Password: Displays password for logging in to Access Point. The password change is not allowed when Access Point is offline/down otherwise the password change is also applied to the Access Point. Note: when Access Point is unauthenticated new password is used to authenticate to the AP and if successful the new password is saved.

  • Group Name: Displays User specified Group name from which the settings are to be applied to configure Access Point.


IP Settings

This page lets you to configure Management IP address settings of managed Access Points. It has the following options:

  • DHCP Client: Enable DHCP client on managed Access Point. Do not enable this unless DHCP server is configured to provide Access Points the same IP address every time. WMS5316 cannot recognize the Access Point as same if its IP address gets modified.

  • IP Address: This is the IP address of managed Access Point. To change it, enter an unused IP address from the address range used on your LAN. This is allowed only when access point is online/healthy.

  • Subnet Mask: Enter the subnet mask value used on your LAN. The default value is 255.255.255.0.

  • Default Gateway: Enter the IP address of the gateway for your LAN.


Click the APPLY button to apply the modified settings.
