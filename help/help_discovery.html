Access Point Discovery Help

From this page you can discover the supported Netgear Access Points that can be managed by WMS5316. For the discovery to work properly, the supported Access points should to be running the base firmware versions listed below.

  • WNDAP330: Firmware WNDAP330_V3.0.4.

  • WNDAP350: Firmware WNDAP350_V2.0.

  • WNAP210: Firmware WNAP210_2.0.8.

  • WG302v2: Firmware 5.2.3 or Firmware 5.2.3.

  • WG103: Firmware WG103_2.0.

  • WN802Tv2: Firmware WN802Tv2_V3.1.2.

  • WG602v4: Firmware V1.1.0.

To obtain the related firmware images, please visit the Netgear support website http://www.netgear.com/support


WMS5316 support two types of discoveries:


Auto Discovery

Auto discovery uses Layer 2 discovery methods and discovers Access points in the same IP network as the WMS5316.


IP Discovery

Use IP discovery to discover APs which are not in the same IP network as WMS5316. You can only search a maximum of 255 IP addresses at a time. Please split up your search if you have APs in multiple networks.


Once discovered, select the APs to be managed by WMS5316 and click the Add button. This will login to each of the Access Points and push the configuration from WMS5316 to the Access Points.


General Note:

  • If there are duplicate IP addresses, WMS5316 can discover model number for only the first one.

  • If the Access Points are not running the base version or above, the Model Numbers cannot be discovered and WMS5316 cannot manage them.

  • Access Points already managed by the WMS5316 will not be shown in the discovery list.

  • Other Netgear devices may also be discovered as unknown.
