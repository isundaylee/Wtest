Security Rogue AP Help


Rogue AP Detection

Unidentified Access Points that use the SSID of a legitimate network can present a serious security threat. Detecting Rogue Access Points involves scanning the wireless environment on all available channels, looking for unidentified Access Points. These settings are applied to all managed Prosafe Access Points.


Rogue AP Detection is enabled by default on managed Access Points.


Import AP List from a file

This option allows you to import a pre-complied list of approved Access Points from a saved file. If you created and saved a list of approved Access Points for the network, then you can save the list into a file and import the file here. This file must be a simple text file with one MAC address per line.


If you choose Merge, the current list will be maintained and the Access Points in the imported list will be added to the approved list as well Known AP List.
