Guest Access


Guest Access settings are useful when you are configuring a public Access Point. You can use it to redirect all HTTP requests to a Web server of your choice, through which you can log each client connection time.


  • Enable Guest Access: Enable this if you want all HTTP (TCP, port 80) requests to be "captured" and routed to the URL you specify in the next field.

  • Redirect Location: Specify Local to redirect to a redirect page on WMS5316. You can specify External and enter a URL for redirecting all HTTP (TCP, port 80) requests.

  • URL:Enter the URL of the web server that you want all HTTP requests to be redirected to. Welcome Page.

  • Select Image File:Specify a jpeg/gif image to upload to WMS5316. This image is set as background for default WMS5316 redirection page; which is displayed when client connects to Access Point with Guest access enabled.

  • Preview:Use this button to display how the redirection page will look.
