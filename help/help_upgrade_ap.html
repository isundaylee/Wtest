Access Point Upgrade Help


Upgrade Firmware


You can install a new version of the Access Point's software using Access Point Upgrade page.


Go to the NETGEAR Web site (www.NETGEAR.com) customer service downloads section to get new versions of the Access Point software for supported models. After downloading an upgrade file, you may need to unzip (uncompress) it before upgrading the Access Point.


IMPORTANT! Once you click Upload do NOT interrupt the process of sending the software to the Access Point and restarting the Access Point.


To upgrade Access Point software:


  • Download the new software for a specific Access Point model to upgrade.

  • If not done automatically, uncompress the downloaded file. If included, read the Release Notes before continuing.

  • Make sure that status of the managed Access Point to be upgraded is healthy. Select the managed Access Point model from the drop down list; the model to be upgraded is not available unless at least one access point of a specific model is managed and can be upgraded.

  • Click Browse.

  • Locate and select the file you just downloaded and uncompressed. From your local hard disk.

  • Click Upload to send the software to the Access Point.
    This loads the new software into the Access Point and causes the Access Point to restart.
    Note: Do not try to go online, turn off the Access Point, shutdown the computer or WMS5316 or do anything else to the Access Point or WMS5316 until the Access Point finishes restarting! When the Test light turns off, wait a few more seconds before doing anything.

  • Check the Firmware Version on upgrade page to verify that your Access Point now has the new software installed.


IMPORTANT! In some cases, such as a major upgrade, you may need to erase the configuration of your Access Point after upgrading it. Refer to the Release Notes included with the software to find out if you need to reconfigure the Access Point. Reconfiguring access points already managed by WMS5316 require only the IP address to be set manually. WMS5316 restores configuration for already managed Access Points running supported software version.
