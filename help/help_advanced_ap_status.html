Monitoring Access Point Status Help


Access Point Status


The Access Point status screen displays a read-only status summary of managed Access Points.


Each Access Point entry specifies:


  • Status: Displays Access Point connection and configuration status.

  • Group: Displays Group configured for Access Point.

  • Access Point: Displays netbios name of Access Point.

  • IP address: Displays management IP address used by WMS5316 to connect to Access Point.

  • Model: Displays Access Point Model.

  • Bg channel: Displays b/g/n mode channel configured on Access Point

  • A channel: Displays a/n mode channel configured for Access Point.

  • Bg clients: Displays count of client stations connected to AP using 2.4GHz channel

  • A clients: Displays count of client stations connected to AP using 5GHz channel.


Click Refresh button to update the displayed status of Access Points. Click Details button to display detailed status of specific managed access point.


Access Point Status Help


The Access Point status screen displays a read-only details of a managed Access Point which includes configuration settings, current Wireless settings, current clients and current traffic statistics.


Access Point Details


The Access Point details section displays information about the Access Point.


  • Access Point Name: The Access Point's NetBIOS name.

  • Model: The managed Access Point's Model.

  • Group: The configured group of managed Access Point.

  • IP address: The IP Address of managed Access Point.

  • Ethernet MAC Address: The Ethernet MAC address of managed Access Point.

  • 2.4 GHz Channel: 2.4 GHz channel configured on access point; set to -NA- if not available.

  • 5 GHz Channel: 5 GHz channel configured on access point; set to -NA- if not available.

  • Load Balancing: Load balancing enable/disable status for Access Point.


Security Profile


The section displays configured and enabled security profiles on the Access Point.


  • Type: 802.11 b/bg/ng or 802.11 a/na mode for security profile.

  • SSID: Wireless Network SSID.

  • Security: Open/wep/wpa/wpa2 mode of security configured for the profile.

  • VLAN: VLAN configured for the security profile.


Client Info


This section displays Access point client stations information.


  • MAC: Wireless MAC of Access Point Client.

  • SSID: Wireless SSID configured on managed Access Point to which client connects.

  • Channel: The channel which client is using to connect.

  • Mode: 802.11 b/bg/ng or 802.11 a/na mode for security profile.

  • Auth: Open/wep/wpa/wpa2 authentication mode of security.


Rogue Access Points


This section displays Rogue/neighboring Access Point detected by managed Access Point.


  • Rogue Access Points Reported: count of Rogue Access Points found

  • Rogue Access Points in same channel: count of Rogue Access Points in configured channel.

  • Rogue Access Points in interfering channels: count of Rogue Access Points in interfering channels.


Statistics


This section displays Ethernet, 802.11b/bg/ng and 802.11a/na mode data traffic statistics for the access points.
Use the REFRESH button to update Access Point statistics and information.
