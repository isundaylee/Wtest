Centralized RF Management


This page lets you change Basic RF Management settings and schedule. Basic RF Management when run tries to optimize the channel allocation for Access Points based on the clients, user data traffic and observed nearby RF environment of Access Points.


Centralized RF Management

Specify the following information in this section:

  • Centralized RF management: Enable Centralized RF Management to let WMS5316 control and change Access Point channel allocation.

  • Client aware RF Management: Enable to stop modifying channel of an Access Point if it has any clients attached.

  • Usage aware RF Management: Enable to stop modifying channel of an Access Point if Access Point is switching more than 1Mbps of Wireless data traffic.

  • Run Channel Allocation at: Specify at what time of the day the channel allocation can modify Access Point RF configuration.

  • Run Channel Allocation every: Specify weekly schedule for running channel allocation.

  • Run Channel Allocation now: Press button to run channel allocation immediately.


Custom RF Settings

  • Mode Settings: Specify radio mode preference for RF Management module to set on the Access Points.

  • 2.4Ghz or 5Ghz band selection: Specify Frequency band preference for RF Management module to set on the Access Points. This configuration only applies to Dual Configurable Access Points. For Dual Concurrent Access Points, both radio mode will be enabled by default.
