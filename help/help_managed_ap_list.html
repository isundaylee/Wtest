Managed Access Point Help


Managed Access Point List

This page displays details of each Access Point controlled by WMS5316. This page lets you edit Access Point connection specific settings. IP settings, Vlan settings password of the Access Points can be changed from the edit page of each AP.


Each Access Point entry specifies:

  • IP Address: Displays IP address of Access Point.

  • Model: Displays the model of Access Point.

  • Name: Displays user specified name of Access Point.

  • Group Name: Displays user specified name of Access Points group.

  • Synchronized: Displays Access Point configuration synchronization status.

  • Status: Displays Access Point connectivity status.


    Value of Status Field

    Implies

    How to correct

    Connected

    Normal Operation. WMS5316 is able to contact the AP and manage it.

    --

    Authentication Failure

    Password Mismatch. The Password configured during synchronization from WMS5316 is not the same as the password of the AP.

    Select the AP and click Edit. In the Edit page, configure the correct password of the AP and click Apply. WMS5316 will try synchronization again with the AP.

    Not Connected

    No Connection. WMS5316 is not able to reach the AP with the configured IP address.

    WMS5316 tries to login to Managed APs every 1 minute. If the error is temporary, then the status will automatically change to connected. If the error is prolonged verify the APs IP address and Network connectivity.


Select the Enable check box to remove corresponding Access Point.


To change the settings of a Access Point, select the Access Point and click the Edit button.
